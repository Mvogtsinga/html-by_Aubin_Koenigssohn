Bienvenue sur My first Webpage by AKtreize

ce ci est reellement la premiere page web que j ai reussi a mettre sur pied grace a mes efforts personel, le support de Elodie qui est deja junior manager dans le domaine du Web developping. c est vraiment pas facille car faut etre patient et perseverant. le monde du Web background est un espace pas facile a maitrisser. Avant le debut de mon BootsCamps chez Code institut,je me sens pres pour le challenge que j espere enrichissant.le developpement de ma premiere page web va me permettre d obtenir l outil necessaire. Deesse a egalement eu son premier cours de coding aujourdhui et a ete satisfait du deroulement.

Service 1

Je continue a apprendre comme il ya de cela une minute j ai appris a place une icon sur ma page. Selon mon instructeur de la video, pour cela je suis d abor aller sur le site cdnjs.com copier taper comme mot de recherche font font-awesome et selectioner le premier resultat. une fois selectionner copier le deuxieme symbole rentrer sur ma page du code l inserer en haut juste au head. apres cette etape tape fontawesone.com et chercher labas le briefcase et selectionner un icon non payant. voila comment j ai reussir a inserrer mon premier icone sur ma premiere page Web. ca fait vraiment du bien de voir son evolution personel. ce qui suivra alors est tres interressant. je vais duplier tout le code services 4 fois afin d avoir 4 possibilites ou 4 different services. c est pas easy mais on continu notre apprentissage du code. bon je vais essayer alors...... 1.... 2.... et trois..... times to try my best on it........
